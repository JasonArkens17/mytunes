#mytunes
(Listen to the songs you love) 

## Stack

Built using Backbone, Jquery, Underscore and Materialize 

####Future Deployment: 
Docker, AWS EC2, AWS Lambda.

####Build tools: 
Bower, ESlint.

####Testing: 
Mocha, Chai.

### Installing Dependencies

From within the root directory:
```sh
bower install
```
